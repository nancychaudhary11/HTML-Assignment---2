# Assignment 2 - Forms and Tables

This assignment demonstrates the use of HTML forms and tables.

### Features

* Created a form to collect user information.
* Used different input fields such as text boxes, radio buttons, checkboxes, and submit buttons.
* Created tables using rows and columns.
* Organized data in a structured tabular format.

### Technologies Used

* HTML5

### Learning Outcomes

* Creating HTML forms
* Using different form elements
* Creating tables with rows and columns
* Organizing and displaying data effectively
